/district/¶

Return a list of districts for the given location and precision.

Requests¶

Resource URL¶

https://api.votizen.com/v1/district/

Parameters¶

Required¶

lat latitude
lon longitude

Optional¶

precision One of: state, city, point, (default city)

Example Queries¶

  • Query for a location in northern Columbus, OH at the default (city) precision: http://api.votizen.com/v1/ballot/?api_key=abcd1234&lon=-82.9907226&lat=40.00237
  • Query for the same location in northern Columbus, OH at point precision: http://api.votizen.com/v1/ballot/?api_key=abcd1234&lon=-82.9907226&lat=40.00237&precision=point

Responses¶

Response Format¶

Responses are in JSON, and contain a list of districts which contain the given location. With point precision, all districts are returned. With lower precisions, only districts which wholly contain the possible location are returned.

200 Response¶

{
    "districts": [
        {
            "id": "406",
            "name": "Congressional District 3",
            "state": "OH"
        }
    ]
}

Non-200 Response¶

{
   "message": "string reason"
}
