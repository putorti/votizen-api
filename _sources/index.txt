Votizen API Documentation
=========================

The Votizen APIs provide access to Votizen's voter and election data. This 
documentation provides general principles for interacting with the Votizen 
APIs (:doc:`introduction`) and detailed descriptions of each of our available 
APIs (:doc:`apis`).
